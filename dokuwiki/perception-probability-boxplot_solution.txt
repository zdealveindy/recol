====== Perception of probability: draw the boxplot (Solution) ======
<code rsplus>
# Read data from the raw version of GitHub repository:
probly <- read.csv ('https://raw.githubusercontent.com/zonination/perceptions/master/probly.csv')

# Enlarge left side plot margin:
par (mar = c (5.1, 10.1, 4.1, 2.1))

# Draw horizontal boxplot:
boxplot (probly, horizontal = T, las = 1, xlab = 'Assigned probability [%]',
         main = 'Perception of Probability')

</code>
